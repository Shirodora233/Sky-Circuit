# 1. 基本概念

## 1.1 名称

| 语言 | 名称 |
|---|---|
| 日文 | フライングサーカス |
| 英文 | Flying Circus |
| 简称 | FC |
| 中文可译 | 空中竞技 / 飞行竞技 / 飞翔竞技 |

Flying Circus 是《苍之彼方的四重奏》世界观中，使用反重力飞行鞋「グラシュ / Grav-Shoes」进行的新兴空中运动。比赛不是单纯竞速，而是结合了：

- 空中竞速；
- 浮标触碰得分；
- 近距离背部触碰对抗；
- 立体空间中的路线选择与战术判断。

动画官网将 FC 概括为：选手穿专用服装和 Grav-Shoes，在纵横可飞行区域中飞行，并在限制时间内争取尽可能多的分数。原作官网也将 FC 选手、竞技类型、Dogfight 等作为专门术语说明。

## 1.2 竞技定位

FC 的特点可以概括为：

```text
空中立体运动 + 浮标竞速 + 背后触碰 + 反重力碰撞 + 战术辅助
```

它和普通竞速运动不同，因为选手不只是抢路线，还可以通过近身 Dogfight 主动攻击对手背后得分；它和普通格斗运动也不同，因为没有传统意义上的血量或击倒，而是以点数和位置优势为核心。

# 2. 世界观与飞行技术基础

## 2.1 反重力子 / Anti-Graviton

原作世界中发现了能够与地球重力相斥的粒子，称为「反重力子 / Anti-Graviton」。这种技术可以小型化，但难以大型化，因此出现的是个人飞行鞋，而不是大规模空飞汽车。

## 2.2 Anti-Graviton Shoes / Grav-Shoes / グラシュ

「アンチグラビトンシューズ」是内置反重力装置的飞行鞋，简称「グラシュ」。它让人可以在空中飞行。

Grav-Shoes 分为：

| 类型 | 用途 |
|---|---|
| 一般用 Grav-Shoes | 日常移动使用，外观接近普通鞋 |
| 竞技用 Grav-Shoes | Flying Circus 比赛专用，更重视飞行性能 |

在设定中，竞技用 Grav-Shoes 还会按选手类型分为：

- Speeder 用；
- Fighter 用；
- All-Rounder 用。

## 2.3 起动键 / Activation Key

Grav-Shoes 不是只打开电源就能飞行。使用者需要说出设定好的起动键，初始起动词为：

```text
FLY
```

起动键可以被改成任意词。

## 2.4 Membrane / メンブレン

Grav-Shoes 启动后，会产生反重力子的膜，称为「メンブレン / Membrane」。

它的作用是：

- 覆盖使用者身体；
- 产生反重力效果；
- 让使用者能自由飞行；
- 与其他 Membrane 接触时产生排斥。

## 2.5 Membrane 的反发 / 反重力排斥

两个启动 Grav-Shoes 的人互相接触时，双方的 Membrane 会相互排斥，因此双方会被弹开。

这一点是 FC 的核心规则来源：

- 背部触碰可以判定得分；
- 触碰瞬间双方会被弹开；
- 弹开方向会影响之后的路线、位置与攻防；
- 滥用反发力的危险动作可能被视为犯规。

## 2.6 Balancer / バランサー

Balancer 是 Grav-Shoes 内置的飞行姿势稳定辅助功能。

作用：

- 稳定飞行姿态；
- 降低失控风险；
- 帮助普通选手维持可控飞行。

部分剧情和技术涉及关闭或调整 Balancer，以换取更高机动性或更强性能，但风险也更高。

## 2.7 Pairing / ペアリング

Pairing 指两个人使用一组 Grav-Shoes 一起飞行。启动时如果与使用者接触，例如牵手，就可以一起起飞。

但当双方的 Grav-Shoes 都处于启动状态时，由于反重力子互相排斥，彼此不能正常牵手。

# 3. 比赛装备

## 3.1 竞技用 Grav-Shoes

FC 比赛使用专门为竞技设计的 Grav-Shoes。相比日常用鞋，竞技鞋更重视飞行性能，外观也更个性化。

按竞技风格分为：

| 竞技鞋类型 | 对应选手风格 | 大致特征 |
|---|---|---|
| Speeder 用 | 速度型 | 最高速强，适合浮标抢点 |
| Fighter 用 | 格斗型 | 短距离机动强，适合 Dogfight |
| All-Rounder 用 | 全能型 | 速度和近战平衡 |

## 3.2 Flying Suit / フライングスーツ

Flying Suit 是专为 FC 设计的飞行特化服装。

设定中它具有安全功能：

- 适合飞行；
- 袖子部分带有类似浮袋的功能；
- 当 Grav-Shoes 故障、选手落入海中时，可以辅助漂浮。

## 3.3 安全环境

FC 常在海面上空进行，因此落水比坠地更安全。场地通常设置在水域上方，既符合视觉表现，也符合安全逻辑。

# 4. 比赛场地

## 4.1 基本场地

标准 FC 场地由空中的浮标构成。

常见形式：

```text
4 个浮标构成一个方形空中赛道
```

浮标之间以线连接，并形成编号线。选手围绕这些浮标进行飞行、抢点与对抗。

## 4.2 标准赛制与简化赛制

| 赛制 | 浮标数量 | 时间 |
|---|---:|---:|
| 标准赛制 | 4 个浮标 | 10 分钟 |
| 简化赛制 | 2 个浮标 | 5 分钟 |

Fandom 补充资料中提到，FC 通常采用 4 浮标、10 分钟规则，也可以采用 2 浮标、5 分钟形式。

## 4.3 场地位置

FC 是户外运动，比赛场地通常位于水域上方数米处。这样可以解释：

- 为什么选手落下后通常是落水；
- 为什么 Flying Suit 有浮力安全设计；
- 为什么观众可以从海岸或平台观看空中比赛。

# 5. 比赛基本规则

## 5.1 对战人数

FC 比赛通常为：

```text
1 对 1
```

两名选手在空中场地中争夺分数。

## 5.2 胜负条件

在限制时间内，得分更高者获胜。

标准设定：

```text
10 分钟内获得更多分数者胜利
```

## 5.3 平局与加时

补充资料中提到：

- 若常规时间结束出现平局，会进入加时；
- 加时阶段仍未分出胜负时，可采用下一分胜负的形式。

具体加时结构在不同整理资料中可能存在表述差异，开发游戏时可简化为：

```text
常规时间平局 → 进入 1 分钟或 5 分钟加时 → 仍平局则金分制
```

## 5.4 中途暂停

补充资料中提到，比赛开始后不允许 Timeout。

## 5.5 男女混合

补充资料中提到，FC 是男女混合竞技，不区分男子组与女子组。

# 6. 得分方式

FC 有两种主要得分方式。

## 6.1 触碰浮标得分

选手沿指定方向飞行，触碰对应浮标后获得分数。

特点：

- 稳定；
- 风险较低；
- 适合 Speeder；
- 形成比赛的基本路线与节奏。

## 6.2 触碰对手背部得分

选手也可以不抢浮标，而是在近距离 Dogfight 中触碰对手背部得分。

特点：

- 技术性强；
- 风险较高；
- 可以打断对手路线；
- 适合 Fighter；
- 是比赛最具观赏性的部分之一。

## 6.3 两种得分的关系

FC 的核心魅力就在于两种得分方式互相制衡：

| 战术方向 | 优点 | 缺点 |
|---|---|---|
| 抢浮标 | 稳定推进，路线明确 | 容易被预测和拦截 |
| Dogfight | 可快速反超，观赏性强 | 失败后可能被弹开，损失路线 |

因此 FC 不是纯速度赛，也不是纯空战，而是两者结合。

# 7. 路线、浮标与 Shortcut 规则

## 7.1 顺时针移动

补充资料中提到，选手围绕浮标顺时针移动。

选手可以：

- 停止；
- 后退；
- 调整路线。

但需要注意：

```text
不能越过前一个浮标
```

这保证了比赛仍然围绕赛道结构展开，而不是完全自由飞行。

## 7.2 领先者限制

补充资料中提到，领先选手在通过第一个浮标后，不能在落后选手追上前继续触碰下一个浮标。

这个规则的作用是：

- 防止 Speeder 一旦领先就无限滚雪球；
- 迫使领先者考虑防守、等待与拦截；
- 给落后者制造追赶与 Dogfight 的机会。

## 7.3 落后者可以跳过浮标

落后选手可以跳过浮标，直接向后续区域移动。

这个行为与 Shortcut 规则相关，目的不是让落后者无条件得分，而是让其获得战术位置，重新进入攻防。

## 7.4 Shortcut / ショートカット

Shortcut 指放弃触碰当前浮标，直接移动到下一条线或下一个区域。

官方动画官网术语说明中，Shortcut 被解释为：

- 放弃当前浮标触碰；
- 移动到下一条线；
- 用来避免比赛变成单纯最高速对决。

Shortcut 后并不是马上可以继续得分。需要满足额外条件：

- 与对手交叉一次；或
- 触碰对手；
- 然后才能触碰之后的浮标。

也可以进行 1 线、2 线、3 线的 Shortcut。

## 7.5 Shortcut 的战术价值

Shortcut 的作用：

1. 让低速选手不至于被 Speeder 完全甩开；
2. 让 Fighter 可以放弃浮标，提前占据拦截位置；
3. 强迫 Speeder 不能只沿最优路线高速巡航；
4. 让比赛从竞速变成路线博弈。

# 8. Dogfight 近身战规则

## 8.1 Dogfight 定义

Dogfight 是 FC 中围绕「夺取对手背部」展开的近身攻防。

核心动作：

```text
接近对手 → 绕到背后 → 触碰背部 → 得分 → 双方弹开
```

原作官网和动画官网都强调，Dogfight 是 FC 的最大看点之一。

## 8.2 背部触碰

Dogfight 的目标不是击倒对手，而是触碰对手背部。

背部触碰成功后：

- 触碰者获得分数；
- 对手背部会出现视觉提示；
- 双方因反重力排斥被弹开；
- 比赛重新进入路线争夺或下一轮近身攻防。

## 8.3 反弹方向的重要性

由于双方接触后会被弹开，触碰本身不只是“获得分数”这么简单，还会改变双方之后的位置。

因此 Dogfight 的高级策略包括：

- 选择能把自己弹向下一浮标的角度；
- 把对手弹离路线；
- 利用接触后的反弹制造下一次进攻机会；
- 避免因错误接触把自己弹到不利位置。

## 8.4 Dogfight 与战斗机术语

Dogfight 一词本来来自战斗机空战中互相争夺背后位置的情况。FC 借用这一概念，用来形容空中选手围绕对方背部进行的高速近身攻防。

# 9. 犯规与危险行为

## 9.1 开局前接触犯规

补充资料中提到，在第一个浮标被触碰前，选手不能接触对手。

这可以理解为：

```text
开局阶段必须先围绕浮标建立比赛流程，不能一开始就强行 Dogfight。
```

## 9.2 滥用反发力

Grav-Shoes 的反发力是 FC 规则的一部分，但滥用它进行危险动作会被视为犯规。

危险行为包括：

- 使用双臂或双腿夹击，让对手在反发力中来回弹动；
- 利用反发力把对手撞向浮标；
- 其他明显可能造成伤害的技巧。

严重情况下可能导致：

- 犯规；
- 失格；
- 禁赛；
- 对手受伤。

## 9.3 Suicida 的特殊性

Suicida 是一种将对手从上方压向水面、让对手行动受限的粗暴技术。动画官网资料说明它虽然粗暴，但属于规则允许的技术，因此不属于反则。

这说明 FC 的规则并非禁止所有强硬对抗，而是禁止过度滥用反发力、造成明显危险的行为。

# 10. 选手、Second 与竞技类型

## 10.1 Skywalker / スカイウォーカー

FC 选手统称为：

```text
Skywalker / スカイウォーカー
```

Skywalker 按竞技风格分为三类：

- Speeder；
- Fighter；
- All-Rounder。

## 10.2 Speeder / スピーダー

Speeder 是速度型选手。

特点：

- 擅长高速移动；
- 主要依靠触碰浮标得分；
- 倾向于避免 Dogfight；
- 急转弯较弱；
- 初速较慢；
- 最高速很高。

战术定位：

```text
拉开距离 → 进入高速巡航 → 连续抢浮标 → 避免近身战
```

## 10.3 Fighter / ファイター

Fighter 是 Dogfight 特化选手。

特点：

- 擅长近距离小范围机动；
- 目标是触碰对手背部得分；
- 初速快；
- 最高速度较低；
- 不适合长距离高速巡航。

战术定位：

```text
切线拦截 → 进入近身 → 抢背后位置 → 触背得分
```

## 10.4 All-Rounder / オールラウンダー

All-Rounder 是速度与 Dogfight 平衡的选手。

特点：

- 介于 Speeder 和 Fighter 之间；
- 没有明显短板；
- 能根据局势切换打法；
- 可以抢浮标，也可以打 Dogfight。

战术定位：

```text
根据比分、位置和对手类型灵活切换策略
```

## 10.5 Second / セコンド

Second 是比赛中通过耳机向选手提供指示和建议的场外辅助者。

FC 允许 Second 的原因：

- FC 是三维立体运动；
- 对手可能出现在 360 度任意方向；
- 选手很容易丢失对手位置；
- Second 的基本任务是帮助选手持续掌握对手位置。

Second 不只是“教练”，更像是：

- 空中导航员；
- 战术观察员；
- 雷达辅助；
- 心理与策略支持者。

# 11. 视觉辅助与观赛表现

## 11.1 Contrail / コントレイル

Contrail 是从 Grav-Shoes 鞋跟部分发出的光迹。

作用：

- 让观众更容易看到选手轨迹；
- 增强高速飞行的视觉表现；
- 区分不同选手；
- 帮助判断路线和技术动作。

补充资料中提到，Contrail 颜色通常与鞋的颜色对应，而且不同选手之间通常避免使用相同颜色。

## 11.2 Point Field / ポイントフィールド

Point Field 是背部触碰得分时出现的视觉提示。

特点：

- 选手在使用竞技用 Grav-Shoes 飞行时，被触碰背部会出现；
- 用于让观众和裁判明确看到背部触碰得分；
- 表现上常为背部的发光标记。

## 11.3 观赛逻辑

FC 高速、立体、空间变化大，因此观赛辅助非常重要：

- 选手尾迹显示飞行路径；
- 背部得分标记显示得分瞬间；
- 浮标和线路让观众理解当前目标；
- Second 的存在也说明比赛空间认知难度很高。

# 12. 基础技术与特殊技术

## 12.1 技术分类

资料中通常把 FC 技术分成：

| 分类 | 说明 |
|---|---|
| 基础技术 | 多数 Skywalker 应掌握，用于速度、高度、路线或防守优势 |
| 特殊技术 | 高水平选手使用，通常需要更强控制能力或特殊条件 |

## 12.2 Field Fly / フィールドフライ

Field Fly 是基础训练，用于掌握正确飞行姿势。

用途：

- 新手训练；
- 姿态控制；
- 熟悉 Grav-Shoes 的基本飞行。

## 12.3 Scissors / シザーズ

Scissors 是一边向前飞行，一边左右急转，形成锯齿状轨迹的技术。

用途：

- 扰乱阻挡者；
- 突破待伏；
- 被追背时让对手被挤到前方；
- 常由 Speeder 使用。

动作结构：

```text
前进 → 左急转 → 右急转 → 反复形成 Z 字路线
```

## 12.4 Low Yo-Yo / ローヨーヨー

Low Yo-Yo 通过下降获得速度，再上升回到路线。

用途：

- 帮助初速慢的 Speeder 获得速度；
- 利用重力加速；
- 防止被对手从背后触碰。

动作结构：

```text
下降 → 加速 → 半途上升 → 恢复路线
```

## 12.5 High Yo-Yo / ハイヨーヨー

High Yo-Yo 是抬头上升、减速，再利用急下降重新加速并抢背后的技术。

用途：

- 让追在背后的对手冲到自己前方；
- 通过失速制造位置交换；
- 之后俯冲加速，反过来攻击对手背部；
- 常由 Fighter 使用。

动作结构：

```text
上升 → 减速 / 失速 → 对手前冲 → 俯冲 → 攻击背部
```

## 12.6 Airkick Turn / エアキックターン

Airkick Turn 是 FC 中的高级技术之一。

表现：

- 像在空中踢了一脚一样；
- 实现急剧方向转换；
- 需要精准控制 Membrane；
- 技术难度高。

用途：

- 急转反打；
- 改变飞行方向；
- 接近对手或返回路线；
- 瞬间制造意外角度。

## 12.7 Kirimomi / キリモミ

Kirimomi 是一边旋转身体一边飞行的技术。

用途：

- 防御性动作较多；
- 被 Dogfight 攻击时，通过旋转让对手难以触碰背部；
- 干扰对手的触背时机。

## 12.8 Reversal / リバーサル

Reversal 是避开位于上方的对手，并反过来取得上方位置的技术。

用途：

- 夺取高度优势；
- 反制压制；
- 改变上下位置关系。

## 12.9 Cobra / コブラ

Cobra 是高难度技术。

表现：

- 飞行中急刹车；
- 很快恢复飞行姿势并加速；
- 姿态像眼镜蛇抬头威吓。

特点：

- 比 Airkick Turn 更高阶；
- 需要更强 Membrane 控制；
- 通常被视为职业级技巧。

## 12.10 Suicida / スイシーダ

Suicida 是从正上方将对手弹向水面的粗暴技术。

效果：

- 让对手被压向海面；
- 阻止对手行动；
- 属于规则允许的强硬技术。

## 12.11 Force：Delta Force / Pentagram Force

Force 系技术通过特定几何轨迹迷惑和困住对手，然后突然向对手背后突进。

### Delta Force

```text
三角形轨迹 → 迷惑 / 困住对手 → 突进触背
```

常由 Fighter 使用。

### Pentagram Force

Delta Force 的进阶版本，轨迹由三角形扩展为五芒星形。

```text
五芒星轨迹 → 更复杂的扰乱 → 突进触背
```

## 12.12 Blaster Roll

Blaster Roll 是一种从背后攻击中逃脱并反击的回避反制技术。

使用条件：

- 对手在自己背后；
- 双方几乎平行飞行；
- 对手即将攻击。

动作效果：

```text
快速侧移 → 脱离背后攻击线 → 绕到对手背后
```

## 12.13 Angelic Halo

Angelic Halo 是极高速飞行形成包围环，让对手难以逃脱的高阶技术。

特点：

- 超高速；
- 形成类似环状封锁；
- 可以限制对手行动；
- 对体力消耗极大；
- 在资料中被描述为极高难度技术。

## 12.14 Sonic Boost

Sonic Boost 是通过操纵 Membrane 获得巨大速度提升的技术。

用途：

- 瞬间加速；
- 抢浮标；
- 突入 Dogfight；
- 拉开距离。

## 12.15 Birdcage / バードケージ

Birdcage 是通过保持高度优势来支配比赛的战法。

特点：

- 选手保持在对手上方；
- 像把对手关进鸟笼一样控制其行动；
- 要求极高技术；
- 利用高度和视野优势压迫对手。

在动画官网说明中，Birdcage 是乾沙希与イリーナ使用的战法，只有极少数人能做到。

## 12.16 Super Package Mold / SuMo

SuMo 是视觉小说部分路线中的技术，用于对抗高度压制类战术。

核心思路：

- 采用倒飞或特殊姿态；
- 使选手能够更清楚地观察上方对手；
- 反制 Birdcage 这种高度支配战术。

缺点：

- 倒飞会带来更强负担；
- 需要极高 Dogfight 熟练度；
- 不适合普通选手。

# 13. 战术结构整理

## 13.1 Speeder 战术

Speeder 的核心是速度与路线。

常见思路：

```text
利用 Low Yo-Yo 或加速进入高速状态
↓
沿浮标路线持续抢点
↓
避免被 Fighter 拖入 Dogfight
↓
必要时使用 Scissors 突破拦截
```

优势：

- 一旦速度起来，很容易连续压制；
- 浮标得分稳定；
- 适合领先后扩大优势。

弱点：

- 急转弱；
- 低速状态危险；
- 被 Fighter 切线拦截后容易陷入 Dogfight。

## 13.2 Fighter 战术

Fighter 的核心是近身与背后位置。

常见思路：

```text
通过 Shortcut 或切线抢占对手路线
↓
逼迫 Speeder 减速或转向
↓
进入 Dogfight
↓
利用 High Yo-Yo / Airkick Turn / Blaster Roll 抢背
↓
触背得分后利用反弹重置位置
```

优势：

- 短距离爆发和转向强；
- 适合落后时反打；
- Dogfight 得分具有逆转性。

弱点：

- 最高速不足；
- 如果无法拉近距离，会被 Speeder 拖死；
- 失误后可能被弹到不利位置。

## 13.3 All-Rounder 战术

All-Rounder 的核心是适应。

常见思路：

```text
对 Speeder：主动截线，增加 Dogfight 比例
对 Fighter：保持路线，避免被拖入近战
对 All-Rounder：根据比分切换策略
```

优势：

- 没有明显短板；
- 能根据局势调整；
- 更适合复杂比赛。

弱点：

- 速度不如 Speeder；
- 近战不如 Fighter；
- 对判断和战术要求更高。

## 13.4 高度优势

FC 是立体运动，高度并不是装饰，而是战术资源。

高度优势可以带来：

- 更好的视野；
- 俯冲加速；
- 对低位选手的压迫；
- Dogfight 中的进入角度优势。

但高度过高也可能导致：

- 路线变长；
- 浮标触碰效率降低；
- 被对手 Shortcut 抢线。

## 13.5 Second 的战术价值

Second 的存在说明 FC 的信息战很重要。

Second 可以帮助选手：

- 追踪对手位置；
- 判断对手是否准备 Shortcut；
- 提醒背后风险；
- 制定赛前策略；
- 在比赛中给予心理支持。

# 14. 可抽象为游戏系统的要点

如果把 FC 规则转化为 Unity 游戏机制，可以抽象成以下系统。

## 14.1 核心系统

| 原作设定 | 游戏系统 |
|---|---|
| Grav-Shoes | 飞行控制器 |
| Membrane 反发 | 碰撞反弹系统 |
| 浮标触碰 | 赛道得分系统 |
| 背部触碰 | 近身锁定 / 背刺判定系统 |
| Second | 雷达 / AI 教练 / 提示系统 |
| Contrail | 尾迹渲染 |
| Point Field | 得分反馈特效 |
| Balancer | 稳定辅助 / 高级模式切换 |
| Shortcut | 路线策略规则 |

## 14.2 最重要的玩法循环

```text
抢浮标路线
  ↕
判断是否 Shortcut
  ↕
进入 Dogfight
  ↕
触背得分或被弹开
  ↕
重新选择路线
```

## 14.3 推荐游戏化规则

为了便于实现，可以把规则简化为：

- 1v1；
- 3 分钟一局；
- 4 个浮标；
- 触浮标 +1；
- 触背 +2；
- 触背后双方弹开；
- 出界或落水重置；
- Speeder / Fighter / All-Rounder 三套参数；
- Second 改成 UI 雷达和提示。

# 17. 整理结论

Flying Circus 的核心不是“飞得最快”，而是：

```text
在三维空间中，通过路线、速度、高度、近身背部争夺和反重力弹开机制，争取更多得分。
```

因此，如果要把它改造成 Unity 原创玩法，最应该保留的是：

1. 4 浮标空中赛道；
2. 触浮标与触背两种得分方式；
3. Speeder / Fighter / All-Rounder 的打法差异；
4. Dogfight 背后位置争夺；
5. 接触后的反重力弹开；
6. Second / 雷达式信息辅助；
7. 高度、路线和 Shortcut 带来的战术变化。

只要这些机制成立，即使不用原作角色和美术，也能做出接近 FC 精髓的原创空中竞技游戏原型。
